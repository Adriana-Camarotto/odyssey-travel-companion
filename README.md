# Odyssey Travel Companion

![License](https://img.shields.io/badge/license-MIT-red.svg)

## Description

The Odyssey Travel Companion application provides tourists with up-to-date information on popular attractions, events, restaurants, and accommodations in the area, so that they can make informed decisions about how to spend their time and money during their trip. Users are able to search for and view detailed information on each item including descriptions and photos.
